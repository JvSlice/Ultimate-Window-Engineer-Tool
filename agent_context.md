# Ultimate Window Engineer Tool — Agent Context

## App
Ultimate Window Engineer Tool / UWE

## Platform
Flutter Web, later possibly Windows desktop.

## Style
Black/green terminal-style UI.

## Main sections
- Convert It
- Fabricate It
- Reference It
- Window Testing Tools

## User preference
When adding features, provide clean full copy/paste code without breaking existing code.

## Important behavior
The app should be modular. New calculators, reference pages, and conversion tools should be easy to add.

## Recent feature examples
- U-value conversion: W/m²·K to BTU/h·ft²·°F
- R-value support
- Thermal expansion calculator under Reference It > Physics
- Material data page with material coefficients, including concrete

## Agent rules
- Inspect code before editing.
- Preserve routes and styling.
- Prefer adding files/modules instead of rewriting large files.
- Give clear file paths.
- Include test steps.

## Search behavior goal
Search should not depend on manually adding super-specific keys for every new item. Prefer a centralized searchable registry, generated index, tags, titles, descriptions, aliases, and route metadata.
